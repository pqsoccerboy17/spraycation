# SPRAYCATION

Knowledge base, sponsorship strategy, and project management for Zuzu Perkal's SPRAYCATION mural tour.

## What is Spraycation?

SPRAYCATION is a female-led traveling mural tour that transforms blank walls into colorful, community-centered public art. Led by Austin-based muralist Zuzu Perkal (@zuzubee), the tour brings art, storytelling, and creative energy to cities across the U.S.

## Project Purpose

This repo supports the **2026 tour planning** -- organizing materials from 2024-2025 tours and building the strategy to secure brand partnerships and funding for a nationwide expansion.

## Structure

- `knowledge-base/` -- Artist profile, tour history, brand collaborations, press coverage
- `sponsorship/` -- Funding strategy, sponsor targets, grant opportunities, outreach tracking
- `deliverables/` -- 2025 wrap deliverables checklist
- `project-plan.md` -- Master checklist tracking all workstreams (persists across sessions)
- `SPRAYCATION 2025/` -- Source materials (docs, photos, media archives)
